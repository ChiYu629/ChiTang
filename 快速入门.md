## 📦 第一步：检查环境

### 1.1 确认Python已安装

打开命令行（Windows按 `Win+R` 输入 `cmd`），输入：

```bash
python --version
```

应该显示 Python 3.8 或更高版本。如果没有，请先安装Python：
- 下载地址：https://www.python.org/downloads/
- 安装时记得勾选 "Add Python to PATH"

### 1.2 进入项目目录

```bash
cd F:\【004】PythonWork（暂不更新）\face_dectecion\deepface_2025
```

### 1.3 检查项目完整性

```bash
python 检查项目.py
```

如果看到 `[成功] 项目完整性检查通过！`，说明一切正常。

## 🚀 第二步：安装依赖

### 方法1：自动安装（推荐）

**Windows用户：**
```bash
双击 "启动系统.bat"
```
脚本会自动检查并安装依赖包。

**Linux/Mac用户：**
```bash
./启动系统.sh
```

### 方法2：手动安装

```bash
pip install -r requirements.txt
```

如果下载速度慢，使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

安装时间：首次安装约需5-10分钟（取决于网速）

## 🎭 第四步：开始使用

### 4.1 摄像头实时检测

1. 点击 "📹 摄像头检测" 选项卡
2. 点击 "启动摄像头" 按钮
3. 允许浏览器访问摄像头
4. 对着摄像头做出不同表情，系统会实时识别

**提示：**
- 确保光线充足
- 保持正面面对摄像头
- 表情要明显一些

### 4.2 图片检测

1. 点击 "🖼️ 图片检测" 选项卡
2. 点击 "📁 选择图片" 按钮
3. 选择一张包含人脸的照片
4. 等待几秒，系统会显示检测结果

**测试图片：**
项目自带测试图片在 `data/pic003.png`

### 4.3 视频检测

1. 点击 "🎬 视频检测" 选项卡
2. 点击 "📁 选择视频" 按钮
3. 选择一个视频文件
4. 系统会在后台处理（可能需要几分钟）
5. 处理完成后，结果保存在 `output/videos/` 目录

**测试视频：**
项目自带测试视频在 `data/video.mp4`

## ❓ 常见问题

### Q1: 摄像头打不开怎么办？

**解决方案：**
1. 检查摄像头是否被其他程序占用（QQ、微信等）
2. 关闭其他使用摄像头的程序
3. 重启浏览器
4. 允许浏览器访问摄像头权限

### Q2: 检测不到人脸怎么办？

**解决方案：**
1. 确保光线充足
2. 使用正面清晰照片
3. 人脸不要太小（建议占图片1/4以上）
4. 避免侧脸或遮挡

### Q3: 安装依赖时出错怎么办？

**解决方案：**
1. 使用国内镜像源（见上文）
2. 检查网络连接
3. 尝试使用管理员权限运行命令行
4. 更新pip：`python -m pip install --upgrade pip`

### Q4: 中文显示乱码怎么办？

**解决方案：**
1. 确保系统有中文字体（Windows自带黑体）
2. 检查 `simhei.ttf` 文件是否存在于项目根目录
3. 如果缺失，从网上下载黑体字体放到项目目录

### Q5: 视频处理太慢怎么办？

**这是正常现象：**
- 视频处理需要逐帧分析，速度取决于视频长度和电脑性能
- 建议先用短视频测试（10-30秒）
- 处理时可以在后台运行，不影响其他操作
