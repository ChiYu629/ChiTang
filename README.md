# 表情识别系统--By季科20260312

一个基于 DeepFace 的实时表情识别系统，支持摄像头实时检测、图片检测和视频检测三种模式。

## 功能特点

- 🎥 **摄像头实时检测**：实时捕捉并分析面部表情
- 🖼️ **图片检测**：上传图片进行表情分析
- 🎬 **视频检测**：批量处理视频文件并生成分析报告
- 🌐 **美观的Web界面**：简洁易用的操作界面
- 📊 **详细报告**：自动生成表情统计报告

## 支持的表情类型

- 😠 生气 (angry)
- 🤢 厌恶 (disgust)
- 😨 害怕 (fear)
- 😊 高兴 (happy)
- 😢 伤心 (sad)
- 😲 惊讶 (surprise)
- 😐 中性 (neutral)

## 环境要求

- Python 3.8 或更高版本
- 摄像头（用于实时检测功能）
- Windows / macOS / Linux

## 快速开始

### 1. 安装依赖

打开命令行（Windows 按 `Win+R` 输入 `cmd`），进入项目目录：

```bash
cd F:\【004】PythonWork（暂不更新）\face_dectecion\deepface_2025
```

安装所需的 Python 包：

```bash
pip install -r requirements.txt
```

> **提示**：如果下载速度慢，可以使用国内镜像源：
> ```bash
> pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
> ```

### 2. 启动系统

运行主程序：

```bash
python app.py
```

看到以下提示说明启动成功：

```
============================================================
表情识别系统启动
访问地址: http://localhost:5000
============================================================
```

### 3. 使用系统

在浏览器中打开 `http://localhost:5000`，你会看到三个功能选项卡：

#### 📹 摄像头检测
1. 点击"启动摄像头"按钮
2. 允许浏览器访问摄像头
3. 系统会实时显示检测结果，在人脸周围绘制绿色边框并标注表情

#### 🖼️ 图片检测
1. 点击"选择图片"按钮
2. 选择一张包含人脸的图片（支持 JPG、PNG 格式）
3. 系统会自动分析并显示检测结果
4. 结果图片会保存在 `output/snapshots/` 目录

#### 🎬 视频检测
1. 点击"选择视频"按钮
2. 选择一个视频文件（支持 MP4、AVI 格式）
3. 系统会在后台处理视频（可能需要几分钟）
4. 处理完成后，结果视频保存在 `output/videos/` 目录
5. 分析报告保存在 `output/reports/` 目录

## 常见问题

### 1. 安装依赖时出错

**问题**：`pip install` 时提示网络错误或下载失败

**解决方案**：
- 使用国内镜像源：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`
- 检查网络连接
- 尝试使用管理员权限运行命令行

### 2. 摄像头无法启动

**问题**：点击"启动摄像头"后没有画面

**解决方案**：
- 确保电脑有可用的摄像头
- 检查摄像头是否被其他程序占用（如 QQ、微信等）
- 允许浏览器访问摄像头权限
- 尝试重启浏览器

### 3. 检测不到人脸

**问题**：上传图片或视频后提示"未检测到人脸"

**解决方案**：
- 确保图片/视频中有清晰的正面人脸
- 光线充足，人脸不要太小或太模糊
- 尝试使用其他图片/视频

### 4. 中文显示乱码

**问题**：界面或结果中的中文显示为方块或乱码

**解决方案**：
- Windows 用户：确保系统中有 `simhei.ttf` 字体（黑体）
- 字体路径：`C:\Windows\Fonts\simhei.ttf`
- 如果没有，可以从网上下载并放到项目根目录

### 5. 视频处理速度慢

**问题**：处理视频需要很长时间

**解决方案**：
- 这是正常现象，视频处理需要逐帧分析
- 处理时间取决于视频长度和电脑性能
- 建议先用短视频测试（10-30秒）
- 可以在后台处理，不影响其他操作

## 技术说明

### 核心技术
- **DeepFace**：Facebook 开源的人脸识别和表情分析库
- **OpenCV**：计算机视觉库，用于图像和视频处理
- **Flask**：轻量级 Web 框架
- **YOLOv8**：高效的人脸检测模型

### 性能优化
- 摄像头模式：每 5 帧分析一次，提高实时性
- 视频模式：每 3 帧分析一次，平衡速度和准确性
- 使用 YOLOv8 检测器，速度快且准确
